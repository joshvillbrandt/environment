# environment

Make a new machine feel like home.

## Setup

Install and setup the shell files as follows:

```bash
sh -c "$(wget https://raw.github.com/joshvillbrandt/environment/install.sh -O -)"
```

### Optional

```bash
# set tty flags (ubuntu)
sudo visudo
#Defaults !tty_tickets
```
